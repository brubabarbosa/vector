#	Vector 

Assignment for the Basic Data Structures I class where we implement a class called vector that follows
the list ADT design and store its elements as a dynamic array.

### Dependencies 

- Cmake
- Gtest


##	Compiling and Execution
	1 - mkdir build
	2 - cd build
	3 - cmake ..
	4 - make 
	5 - ./run_tests
